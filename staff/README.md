This is your main screen for recording in-store sales:

1. **Search for Product**
   - Use the search box to find products by name or SKU
   - Or filter by category using the dropdown

2. **Add to Cart**
   - Tap any product to add it to your cart
   - Adjust quantities using + and - buttons
   - Remove items with the trash icon

3. **Complete Sale**
   - Enter customer name (optional)
   - Select payment method: **Cash** or **M-Pesa** only
   - If M-Pesa, enter the transaction code
   - Add any notes (optional)
   - Tap "Confirm Sale"

4. **Done!**
   - Inventory updates automatically
   - Sale is recorded
   - Receipt number generated

## Troubleshooting

**Can't login?**
- Check your internet/WiFi connection
- Make sure XAMPP MySQL is running
- Contact admin if password needs reset

**Product not found?**
- Product may be inactive
- Contact admin to activate it

**Low stock warning?**
- You can still sell remaining items
- Inform admin to restock

**Sale failed?**
- Check if product has sufficient stock
- Verify payment method is selected
- Try again or contact admin
